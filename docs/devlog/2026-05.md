### 문제 / 해결

- 문제:
  - 게시글 목록 조회 응답이 배열이 아니라 객체로 들어와 `postList.map is not a function` 에러가 발생했다.

- 해결:
  - 백엔드 공통 응답 구조가 `ApiResponse` 형태임을 확인하고, 실제 목록 데이터는 `response.data.data`에서 꺼내도록 수정했다.

- 문제:
  - React에서 Spring Boot API 호출 시 CORS 에러가 발생했다.

- 해결:
  - Spring Security 설정에 CORS 설정을 추가하고, React 개발 서버 `http://localhost:5173` origin을 허용했다.

- 문제:
  - 게시글 작성 요청에서 `categoryId`가 null로 전송되어 `400 Bad Request`가 발생했다.

- 해결:
  - 카테고리 목록 응답 필드가 `id`임을 확인하고, select option의 value를 `category.id`로 수정했다.
  - 게시글 작성 요청 시 선택한 값을 `categoryId`로 전달하도록 수정했다.
